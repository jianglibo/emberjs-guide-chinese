# Ember的模板系统

对于多语言使用者来说，不要将Ember的模板系统与其它全功能的模板相比。不是Handlebar无法达成这些功能，而是多余的功能只会引发滥用和混乱。

比如java的Freemark或者velocity，写得复杂的情况下，会分不清到底是模板里面嵌入代码，还是代码里面嵌入模板。也不要期望像Perl一样有嵌套的引用。
